### CS 5630 / CS 6630

The 2016 Visualization website is built with [Jekyll](http://jekyllrb.com).

#### Ubuntu Prerequisites

```ShellSession
$ sudo apt-get install ruby-dev
```

#### Setup

Requires Jekyll 3 or later.

```ShellSession
$ gem install jekyll
```

#### Generate and/or Serve Site

```ShellSession
$ jekyll serve 
```

#### View Site

```ShellSession
$ open http://0.0.0.0:4000/2016/
```
